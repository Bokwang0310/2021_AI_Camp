# 2021 AI Camp

## 주제

영화를 관람하는 관객의 표정과 감정 변화를 분석해 영화의 흥행 여부를 예측할 수 있는 지표 도출

## 동기

재밌는 영화일수록 관객의 감정 변화가 많이 일어날 것이고 그 감정 변화가 표정을 통해 드러난다면 관객의 표정을 분석해 영화의 흥행 여부를 예측할 수 있지 않을까
